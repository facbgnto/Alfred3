# Decisiones de arquitectura

Las decisiones se registran como ADR individuales en [`docs/adr/`](adr/),
uno por decisión, usando [`0001-template.md`](adr/0001-template.md) como
base. Este archivo es solo el índice — no dupliques el contenido aquí.

| ADR | Título | Estado |
|---|---|---|
| [0001](adr/0001-template.md) | Plantilla | Propuesto |

Al agregar una decisión nueva:

1. Copia `adr/0001-template.md` a `adr/NNNN-titulo-corto.md`.
2. Completa contexto, opciones consideradas, decisión y consecuencias.
3. Enlázala en la tabla de arriba.
